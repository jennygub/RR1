Start Experiment

The objective of the experiment: Evaluate applications creation using Base44 - the results of vibe coding using Base44 and user experience with the platform.

Using the business case description provided, you are required to design and implement an application using Base44.

After creating the initial version of the application, you will be asked to evaluate three expected functionalities. For each functionality:

1. Indicate whether Base44 Succeeded, Partially Succeeded, or Failed in implementing it.

2. Provide a clear justification for your assessment.

3. After evaluating all functionalities, interact with Base44 to refine and improve the implementation to support a functionality that did not succeed after the initial implementation.

Upon completion of all questions and before the submission of the form, you will need to include:

* The link to your application

* The full interaction log with Base44 in a file

* Any additional materials in a zip file (optional)
