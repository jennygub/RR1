Vibe Coding – From Natural Language to Working Apps

Task:

Create, evaluate, and improve an application using the Base44 platform.

Detailed instructions:

  1. Using the provided business case description, implement an application in Base44.
  2. After creating the initial version of the application, evaluate the given expected functionalities. For each functionality:
    1. Indicate whether Base44 Succeeded, Partially Succeeded, or Failed in implementing it.
    2. Briefly justify your assessment.
  3. Interact with Base44 to refine and improve functionalities whose implementation was only partially successful or failed.
  4. Complete all fields in the accompanying form. In particular:
    1. Provide a link to your application.
    2. Upload the interaction with Base44 by selecting all text of the conversation in the chat history including both prompts and responses, copying it, and pasting it into a text or Word file.
    3. Upload any additional materials, if applicable, such as interactions with LLMs used to create the prompts, in a zip file.
    4. Complete the usability and ease-of-use questionnaires.
